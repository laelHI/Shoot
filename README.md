# Shoot

A simple shooting game where you shoot arrows at a stump to increase your score. Careful not to hit any previous arrows!

# Features:

Simple Shooting game with:
- arrows represented as lines
- a rotating stump to hit
- real-time scoring and record keeping
- ... and you can play it all with one key!
